Crypto World

A little about me & the project

The Crypto project:

in this project i use Api of CoinGecko to display over 12,000 types of crypto currencies.
you can see additional information by clicking the coin "more info", and create a list of (up to 5) coins to track. There is also an option to search specific coin by name or id, important in this huge amount of coins.
for this project i use HTML,CSS & JS. and also libraries like jQuery & bootstrap

About me:
